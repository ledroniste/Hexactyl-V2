
# HEXACTYL

Advanced Hosting Panel built on Pterodactyl.

## Features

- Credit System
- Integrated Shop
- Auto Server Provisioning
- Multi-location Nodes
- Real-time Statistics
- Ticket System
- Anti-DDoS Monitor
- Auto Wings Deployment
- Docker Production Ready

## Requirements

- Debian 12
- 4GB RAM minimum
- Docker

## Installation

```bash
bash install/install-hexactyl.sh
```

Access panel on:
http://SERVER-IP:8080
"# Hexactyl-V2"
